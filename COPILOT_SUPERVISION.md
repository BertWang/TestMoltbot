## 🚨 問題分類和響應

### 🔴 嚴重問題 (立即介入)
**症狀**: 構建失敗、類型錯誤、核心功能破損

**響應**:
1. 立即通知 Clawdbot
2. 提供修復建議
3. 檢查是否需要回滾
4. 協助快速修復

**例子**:
```
❌ npm run build 失敗
❌ npm run tsc 有 error
❌ 組件無法渲染
```

### 🟡 重要問題 (今天解決)
**症狀**: 功能不完整、設計不一致、測試失敗

**響應**:
1. 記錄在 PR review 中
2. 要求修改
3. 確認修改後再合併

**例子**:
```
⚠️ 鍵盤導航不完整
⚠️ 響應式設計有問題
⚠️ 樣式不符合 Digital Zen
```

### 🟢 輕微問題 (後續優化)
**症狀**: 代碼風格、註釋、文檔

**響應**:
1. 提出建議
2. 可以接受（有改進空間）
3. 記錄為技術債
4. 後續版本改進

**例子**:
```
💬 變量命名可以更清晰
💬 需要添加更多註釋
💬 可以提取為常量
```
